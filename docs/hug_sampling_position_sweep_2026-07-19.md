# HUG 位置与多候选离散度实验（2026-07-19）

## 距离定义

- 腕到点距离：HUG 输出的 MANO wrist 经相机外参和当前标定变换到世界系后，到圆柱中心的欧氏距离。
- 候选平移差：同一次 RGB-D 输入、同一批8个 HUG 候选的 wrist 世界坐标两两欧氏距离。
- 候选旋转差：候选四元数两两之间的最短旋转角，计算为 `2*acos(abs(q1·q2))`。
- 完整 IK：实际 Inspire base 进入 pre-grasp 和 grasp 的40 mm位置容差并执行闭手、抬升。

## 结论

`(-0.12, 0.32, 0.86) m` 是目前最有希望的位置：新增11次（网格1次加固定10次）中，
完整 IK 3次，其中1次确实抓起超过30 mm。HUG并非始终表现为相同方差：稳定批次的8个
候选只相差约10 mm/4°，离群批次可达到200 mm/65°。当前只按“腕到物体距离接近
140 mm”选候选，无法排除旋转离群和对G1跟踪不友好的 pre-grasp。后续候选选择至少应
加入批内稳健离群剔除、旋转一致性以及在线轨迹跟踪代价。
