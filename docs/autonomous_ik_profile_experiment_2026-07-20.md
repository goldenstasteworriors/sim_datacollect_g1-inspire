# 自主抓取 IK 配置与遥操作 IK 对照（2026-07-20）

## 修改内容

保留 Unitree xr_teleoperate 的 G1_29 Pinocchio/CasADi/IPOPT 模型，但新增自主配置：

| 项目 | 上游 teleop | autonomous |
|---|---:|---:|
| 位置平方误差权重 | 50 | 250 |
| 旋转轴角平方误差权重 | 1 | 0.3 |
| 关节正则 | `0.02*||q||²`，偏向全零 | `0.002*||normalized(q-center)||²` |
| 相对当前关节平滑 | 0.1 | 0.03 |
| 优化器硬限位 | xr_teleoperate URDF | URDF与Isaac 0.9软限位交集 |
| 求解后4帧移动平均 | 有 | 无 |

目标函数仍保留完整位置和姿态，并非位置-only。自主配置的设计目标是先保证自主规划给出的
笛卡尔 waypoint 可执行，同时在剩余自由度内逼近 HUG 姿态。

## 运动前静态端点检查

每个 episode 在运动开始前使用独立 IK 实例：

1. 从当前关节状态直接求 pre-grasp；
2. 以前一步静态解作为初值直接求 grasp；
3. 用 Pinocchio FK 和 Isaac frame correction 回算位置、姿态残差；
4. 记录 `solver_success`、位置误差、旋转误差和距 Isaac 软限位的最小余量。

`within_10mm_15deg` 表示位置不超过10 mm且姿态不超过15°。数值求解成功与这个严格
6D标准分开保存，避免把“优化器返回折中解”写成精确有解。

## 20次对照配置

- 圆柱：`(-0.21, 0.28, 0.86) m`；
- pre-grasp 外推30 mm；
- 每次重新采集 RGB-D，HUG 生成8候选；
- autonomous 与 teleop 各20次。

## 静态端点结果

| 指标 | autonomous | teleop |
|---|---:|---:|
| pre-grasp 求解器返回 | 20/20 | 20/20 |
| pre-grasp 满足10 mm/15° | 0/20 | 0/20 |
| pre-grasp 位置残差均值 | 4.4 mm | 63.6 mm |
| pre-grasp 姿态残差均值 | 17.4° | 20.8° |
| grasp 求解器返回 | 20/20 | 20/20 |
| grasp 满足10 mm/15° | 20/20 | 0/20 |
| grasp 位置残差均值 | 1.0 mm | 28.1 mm |
| grasp 姿态残差均值 | 4.0° | 14.4° |
| grasp 最小 Isaac 关节余量 | +0.0002 rad | -0.0239 rad |

autonomous 的 pre-grasp 都能将位置控制在4.2–4.7 mm，但姿态为15.8–18.4°，因此没有
通过人为设置的15°严格线；这是位置优先权衡的预期结果。grasp 则全部达到0.7–1.9 mm、
3.1–8.8°，20次均通过严格6D端点标准。

teleop 的负关节余量说明其 URDF 内的折中解会越过 Isaac 0.9软限位，实际下发时还会被
Isaac 裁剪。这是此前静态求解返回、动态末端却不到位的重要原因。

## 动态执行与抓取结果

| 指标 | autonomous | teleop |
|---|---:|---:|
| pre-grasp 实际到达 | 20/20 | 8/20 |
| grasp 实际到达 | 20/20 | 4/20 |
| 完整IK路径 | 20/20 | 4/20 |
| 抬升不少于30 mm | 14/20 | 3/20 |
| 严格数据质量成功 | 1/20 | 1/20 |

autonomous 唯一严格成功样本抬升91.5 mm，最大线速度0.105 m/s、最大角速度
0.961 rad/s，均通过当前质量阈值。14个抓起样本中的其他样本主要因物体角速度超过
1 rad/s而失败，而不是 IK waypoint 失败。

## 如何解释结果

自主配置已经基本解决此位置的 waypoint 到达问题：pre-grasp和grasp均由原来的低比例
提升到20/20，抓起由3/20提高到14/20。剩余主要问题已经转移为闭手、接触冲击和抬升
阶段的物体角速度。

两批 HUG 目标不是严格配对，同序号 grasp 位置平均相差65.6 mm、最大177.2 mm；因此
不能把每一对 episode 当成相同输入的 A/B。不过静态残差、Isaac 限位越界和动态到达率
三个证据方向一致，且差距远大于样本波动。
